# Hacker Chat

Minimal terminal-style chat app with Matrix green theme and sound effects.

## Run

```bash
pip install -r requirements.txt
python app.py
```

Open: `http://localhost:5000/hckr_chat_`

## Features

- Real-time chat
- Terminal typing sounds
- Matrix green theme
- ASCII art banner
- Typing indicators
- Simple commands: `/help`, `/users`, `/nick`, `/me`

That's it.
